### Installazione

1. Clona la repo
   ```sh
   git clone https://github.com/TUO_NOME/counter.jsbasics.git
   ```
3. Installa i pacchetti NPM
   ```sh
   npm install
   ```
4. Inserisci la tua API su `config.js`
   ```js
   const API_KEY = 'LA TUA API';
   ```
